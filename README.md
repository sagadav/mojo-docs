# _docs

Внутренняя документация проекта Mojo Education.

## Структура

| Папка | Описание |
|-------|----------|
| `attestat/` | Аттестаты: потоки, ERD, шаблоны полей |
| `bugs/` | Разборы багов и тикетов |
| `marks/` | Оценки: потоки выставления, критерии |
| `memory/` | Заметки сессий и память агента |
| `mobile/` | Мобильное приложение: API, встречи |
| `onboarding/` | Онбординг агентов и разработчиков |
| `ops/` | Деплой, Kubernetes, smoke-тесты |
| `reports/` | Отчёты: видимость и роли |
| `school/` | Школьный модуль: долги, чат, роли |
| `scripts/` | Утилитарные скрипты (PS1) |
| `sql/` | SQL-миграции и разовые скрипты |
| `store/` | UI-компоненты (combobox и др.) |
